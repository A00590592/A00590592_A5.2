# Code for Activity A5.2

This repository contains the code developed for Activity A5.2.

It includes one Python script:

- computeSales.py


## Project Structure

The project is organized using the following directory structure:

A5.2/

 ├─ source/

 ├─ tests/
 
 ├─ results/
 
 └─ pylint/


## Folder Description

source/
Contains the Python source code for the program.
The script must be executed from this directory using the command line.

tests/
Contains the input JSON files used to validate program behavior.
This includes:
- Product price catalogue files
- Sales record files

results/
Stores the output files generated by the program execution.
Each result file is automatically named based on the sales input file.

pylint/
Contains the pylint and flake8 analysis reports generated for the program.


## Program Description

computeSales.py calculates:

- TOTAL_COST
- IGNORED_SALES
- TIME_ELAPSED_SECONDS

The program:
- Is invoked from the command line
- Receives two JSON files as parameters:
  1. Product price catalogue
  2. Sales record
- Prints results to the screen
- Writes results to the results/ folder
- Ignores invalid sales records and counts them


## Execution Notes

The program must be executed from the source/ directory.

Usage:

python computeSales.py priceCatalogue.json salesRecord.json

Example:

python source/computeSales.py tests/ProductList/ProductList.json tests/TC3/TC3.Sales.json
